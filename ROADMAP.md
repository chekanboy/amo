# Roadmap — alfa.crm dashboard

## ✅ Сделано
- Миграция монолита на Vite + модульная структура (23 модуля)
- Воронка с потерями между этапами + подсветка "УЗКОЕ МЕСТО"
- Наглядное сравнение периодов (штуки + % + период + инверсия для отказов)
- Таблица эффективности источников с оценкой качества
- Drill-down: клик по цифре → ссылки на сделки в AmoCRM
- Гео: переключатель Дерево/Таблица + поиск + сортировка
- Светлая/тёмная тема, пароль-gate, мобильная адаптация

## 🔜 В работе / очередь (по рекомендациям аналитика)

- [ ] Рек.8 — Расширить drill-down на остальные блоки (частично сделано)

## 📌 Известные особенности (не баги)
- Воронка накопительная: лид на этапе X считается во всех этапах 0..X
- Календарные периоды (не последние N дней): "эта неделя" = Пн–сегодня
- Источники: приоритет поля "(utm_source) Сводный" (чистые значения)
- Calltouch: даты только DD/MM/YYYY, иначе HTTP 500
